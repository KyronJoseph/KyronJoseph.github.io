Difficulty

Unlike standard action games that are very responsive, rpgs like this require very careful planning in order succeed at the game these games are not difficult by nature but by how the game is percieved

For a new player these games can feel overwhelmingly difficult cause the player is punished for small mistakes
the best way to get better at these games is to take things slow and plan out everymove and action as if every step counts
