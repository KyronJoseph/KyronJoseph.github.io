Asking for Help

For players who are heavily struggling these games feature a co-op mode where can work with either a friend or a random person to help you achieve your set goal in game this feature is exclusivly unqiue to these games meant to ecourage players to work together

How to call for Help

every game has a specific item or game state that needs to be achieved so that you can call a player in usually by seeing a summon sign or their name on a list your cooperator either layed down a sign or answered some call from an eaily obtained in game item

Encrouragement
it is highly recommended for those struggling to reach out for help sometimes specific challenges cannot be overcome alone this system is in place to encourage reaching out
